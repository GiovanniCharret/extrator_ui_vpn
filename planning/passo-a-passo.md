# PASSO A PASSO PARA EXPORTAÇÃO

1. Entre em `Z:\LNC\LNC.exe`
2. Entrar em "Relatórios", conforme imagem `manuais/passo1.jpg`
3. No painel da direita, entrar em "Projetos Executados", dentro do sub-menu, conforme imagem `maunais/passo2.jpg`
4. Vai se abrir um painel no grid na parte central da imagem
5. Percorrer via `tab` ou ponteiro o menu até "Programa" (o foco é na cor azul), conforme imagem `manuais/passo3.jpg`
6. Seta para baixo para trocar para o próximo item do ponteiro, conforme imagem `manuais/passo4.jpg`
7. Em "Visualizar" (o botão ganha um contorno quanto o ponteiro passa por ele), conforme imagem `manuais/passo5.jpg`
8. Um arquivo de visualização imprimível em .pdf é carregado (enquanto lê o pdf, o ponteiro muda para um ícone de ampuleta com um "SQL" escrito abaixo), conforme imagem `manuais/passo6.jpg`
9. Clique no ícone de configuração da impressora (o botão ganha um contorno quanto o ponteiro passa por ele), e certifique-se que o texto escrito é Microsoft Print to PDF, conforme imagem `manuais/passo7.jpg` e `manuais/passo8.jpg`
10. Salvar o conteúdo em `output/output.pdf`. Há uma barra de progresso que precisa finalizar, antes retornar para a tela inicial (item 2 desse menu), conforme imagem `manuais/passo9.jpg`
11. Se já tiver `output/output.pdf`, é necessário clicar na janela de confirmação de sobreposição, conforme imagem `manuais/passo10.jpg`
